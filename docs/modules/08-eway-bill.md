# Module 08 — E-Way Bill

## Overview
Full operational module for generating and managing E-Way Bills.

## KPI Strip
Total EWBs, Active, Expiring Today, Cancelled, Road/Rail/Air/Ship breakdown

## Bar Chart
EWBs generated per day for the selected month

## Recent Activity
Last 3 EWB records

## Main Table Columns
EWB No | Date | Party | From → To | Amount | Mode | Status | Validity | Actions

## Row Actions
- Extend Validity
- Cancel E-Way Bill
- View PDF
- Share

## Drawer (Row Click)
- EWB Number + QR
- Consignor / Consignee details
- Item details (HSN, Qty, Value)
- Transport details (vehicle, mode, distance)
- Validity period
- Status timeline

## Status Badges
- Active (green)
- Expiring Soon (yellow)
- Expired (red)
- Cancelled (grey)
