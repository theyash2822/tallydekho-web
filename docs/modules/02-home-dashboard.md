# Module 02 — Home Dashboard

## Overview
First page after login. Gives a bird's-eye view of the business.

## Sections
1. **KPI Cards Row**: Total Sales, Total Purchase, Net Profit, Cash Balance, Receivables, Payables
2. **Sales vs Purchase Bar Chart**: Last 6 months, grouped bars
3. **Top 5 Customers**: Table with customer name + revenue
4. **Quick Actions Panel**: Create Sale, Create Purchase, Create Expense, Record Payment
5. **Alerts Strip**: Overdue invoices, GST filing deadlines, low cash alerts
