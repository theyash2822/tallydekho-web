# Module 12 — Ledgers

## Overview
Full ledger management — list, detail, transactions, documents.

## Top Section (Two-column)
- Left (60%): KPI row — Total Ledgers, Active, Zero Balance, Negative Balance
- Right (40%): Recent ledger activity (from Audit Trail)

## Ledger List Table
- Filters: Group, Balance (Positive/Negative/Zero), Status, Search (Name/Alias/GSTIN/PAN)
- Columns: Name | Group | Opening Balance | Closing Balance | Dr/Cr | Status
- Sort: Name A-Z, Balance high→low, Group

## Ledger Detail Drawer (4 tabs)
### Tab 1: Transactions
- Filterable ledger day book for this ledger
- Columns: Date | Voucher No | Type | Dr | Cr | Balance
- Click → opens voucher PDF

### Tab 2: Summary
- Opening balance, Period Dr/Cr totals, Closing balance
- Period movement chart (line/bar)
- Ledger group hierarchy path

### Tab 3: GST Ledger (only if GSTIN exists)
- GSTIN, Registration Type, Place of Supply
- GST voucher mapping table

### Tab 4: Documents
- All linked docs: Sales Invoices, Purchase, Credit Notes, Payments, Receipts
- Columns: Date | Doc No | Type | Amount | Action

## Navigation Shortcut
From any module, clicking a party/customer name opens this drawer directly.
