# Module 04 — Receivables & Payables

## Overview
Unified hub for tracking money owed to/by the business.

## Layout
- Toggle: Receivables ↔ Payables (changes all content)
- Top: KPI row (dynamic per mode) + two-column overview
- Bottom: 3 tabs

## Tabs
1. **Customer/Vendor-wise** (default): Party list with outstanding + risk badge
2. **Invoice/Bill-wise**: Individual invoice/bill table
3. **Ageing Analysis**: Buckets — 0-30, 31-60, 61-90, 90+ days

## Drawer (Party)
- Header: Name, contact, total outstanding, risk badge
- Ageing blocks
- Invoice/bill list
- Actions: Send Reminder, Record Receipt/Payment, Share PDF, View Ledger

## Receivables KPIs
Total Outstanding, Overdue, Due This Week, Average Collection Days

## Payables KPIs
Total Outstanding, Overdue, Due This Week, Upcoming Payments
