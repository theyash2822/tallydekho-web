# Module 13 — AI Insights

## Overview
AI-powered analytics and recommendations across all modules.

## Top: Insight Capsules Row
Horizontal scrollable, color-coded capsules:
- Green: positive trends ("Sales up 14% vs last month")
- Red: alerts ("Expenses rose 9% unexpectedly")
- Amber: warnings ("₹1,20,000 stuck in overdue receivables")
- Blue: info ("Purchase value below historical average")

Each capsule: insight text + data source + drilldown link

## 6 AI Sections

### 1. Financial Insights
- Profit trend + prediction (next 30/60 days)
- Cashflow Health Score (0–100)
- Expense red flag detection
- Charts: Profit line, Expense overrun heatmap

### 2. Sales Insights
- Fastest growing / declining customers
- AI-predicted sales next 30 days
- Order patterns ("Monday spikes")
- Charts: Sales forecast, Customer contribution donut

### 3. Purchase Insights
- Vendor dependence risk
- Price fluctuation detection
- Charts: Vendor scatter, Cost trend

### 4. Receivables Insights
- Customers likely to delay
- Expected collection this week
- Risk scoring (Low/Medium/High)

### 5. Inventory Insights
- Over/under-stock warnings
- Deadstock risk list
- AI reorder suggestion (Qty + Date)

### 6. Compliance Insights
- GSTR-2A mismatches predicted
- Return filing delays
- "Actions before filing" checklist
