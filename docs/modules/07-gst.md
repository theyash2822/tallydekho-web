# Module 07 — GST Compliance

## Overview
Full GST return management and reconciliation dashboard.

## KPI Row
GSTR-1 Status, GSTR-3B Status, Unmatched Entries, Pending IRNs, ITC Available

## Sub-tabs
1. **GSTR-1** (Outward Supply): Invoice No, Customer, Taxable Value, CGST, SGST, IGST, POS, Status — Export JSON/CSV
2. **GSTR-2A Reconciliation**: Supplier, Invoice, Date, Value, Match/Unmatch, Reason, Accept/Reject actions
3. **GSTR-3B Summary**: Section-wise tax summary (3.1/3.2), ITC, Net GST
4. **GSTR-4** (Composition): Quarter, Outward Value, Tax Liability, Rate, Status
5. **GSTR-6** (ISD): Supplier GSTIN, Invoice, Distributed IGST/CGST/SGST
6. **GSTR-9** (Annual): Consolidated sections 6A/6B/6C, ITC, differences

## GSTR-2A Features
- Match/Unmatch badges (green/red/yellow)
- Mismatch filters
- Accept / Reject reconciliation buttons
- Reason codes for mismatches
