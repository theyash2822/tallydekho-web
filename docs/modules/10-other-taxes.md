# Module 10 — Other Taxes

## Overview
Covers TDS, TCS, Import Duty, Export Duty, Excise, VAT, Cess.

## Tax Type Tabs
TDS | TCS | Import Duty | Export Duty | Excise | VAT | Cess

## KPI Strip (changes per tax type)
- **TDS**: Deducted, Remitted, Pending Pay, Late Fee
- **TCS**: Collected, Remitted, Pending Pay, Late Fee
- **Import Duty**: Assessed, Paid, Pending, Interest
- Others: similar pattern
