# Module 03 — Cash & Bank

## Overview
Tracks all cash and bank transactions. Includes reconciliation.

## Layout
- Top: Two-column overview (KPIs + recent transactions + account details card)
- Bottom: 3 tabs

## Tabs
1. **Cash Register**: Date, Ref, Description, Dr, Cr, Balance — filterable
2. **Bank Register**: Same + Status badges (Cleared / Pending / Reversed)
3. **Reconciliation**: Two-pane (Bank Statement LEFT | Books RIGHT), auto-match, summary strip

## Drawer
Row click → slide-in drawer (480px) with full voucher details

## Key Features
- Auto-match rules: exact amount, ±1 day tolerance, UTR/reference match
- Reconciliation summary: Total / Matched / Pending / Difference
- Create voucher from reconciliation if no match found
