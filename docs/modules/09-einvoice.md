# Module 09 — E-Invoice (Dashboard Only)

## Overview
Read-only compliance dashboard for IRN (Invoice Reference Number) status.
⚠️ No IRN creation/cancellation here — all IRN actions stay in Sales Register.

## KPI Strip
IRNs Generated, IRN Errors, IRNs Pending, Success Rate %

## Daily Trend Bars
- One bar per day (scrollable horizontally)
- Color intensity increases with more errors

## Error Leaderboard
Top 3 error types (e.g., "Invalid GSTIN", "Schema validation failed", "Duplicate IRN")

## Recent Activity List
- IRNs generated today
- Retry successes
- IRNs with errors (link → go to Sales Register)

## Note
This page is informational only. The "Generate IRN", "Cancel IRN", and "Retry" actions live inside the Sales Register IRN column.
