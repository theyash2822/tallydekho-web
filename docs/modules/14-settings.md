# Module 14 — Settings

## Overview
Global app configuration. Two-panel layout (sidebar 25% + content 75%).

## 5 Groups

### 1. Account & Organization
- **Profile**: Name, email, phone, avatar, password — Save / Delete Account (OTP-confirmed)
- **Company Info**: Name, GSTIN, PAN, address, logo, bank details
- **License**: Plan card, Notification Credits card, Team Seats card + Purchase History

### 2. Preferences
- **Language & Region**: Language, timezone, date format
- **Currency & Number Format**: Currency symbol, decimal places, number system (Indian/International)
- **Voucher Configuration**: Per-type config (Sales Invoice, Purchase, Quotation, etc.) — prefix, numbering, fields

### 3. Notifications
- **Channels & Quiet Hours**: Email/SMS/WhatsApp toggles + credits balance + quiet hours
- **Low Stock & Expiry Alerts**: Threshold, frequency, delivery channel
- **Compliance Reminders**: GST, E-Invoice, E-Way Bill, Other Taxes — days-before + channel
- **Payment Reminders**: Rules list (collapsible cards), min balance slider

### 4. Integrations
- **Tally ERP Sync**: Pairing flow (Enter code → Pairing → Paired state), synced companies, Unpair
- **Bank Feeds**: Connected banks list + Add Bank modal
- **E-Way Bill**: Connect NIC account, token status, logs
- **E-Invoice**: Connect NIC account, token status, IRN quota, logs

### 5. Contact & Information
- **About & Versions**: Product version, legal links (Terms/Privacy/OSS), environment info, debug tools
- **Data Security**: Encryption, cache, network, data retention settings
- **Help Center**: FAQ chatbox with suggestion pills ("How to pair with Tally?")
