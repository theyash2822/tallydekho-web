# Module 11 — Audit Trail

## Overview
Complete log of all voucher activity — creates, edits, deletes — across the system.

## KPI Row
Total Entries, Created, Modified, Deleted, Pending Approvals

## Bar Chart
Activity per day for selected period (scrollable)

## Latest 5 Entries
Quick list of most recent audit events

## Tabs
1. **Day Book** — all entries by all users
2. **My Entries** — filtered to current user only

## Table Columns (both tabs)
Date | Voucher No | Voucher Type | Ledger | Dr/Cr | Amount | User | Action (Created/Modified/Deleted)

## Filters
- Voucher Type (All / Sales / Purchase / Expense / Payment / etc.)
- Dr/Cr toggle
- Status (All / Pending)
- Date range

## Row Actions
- View Voucher → opens side drawer
- Multi-select → Bulk Share / Export (PDF, CSV, XLSX)
